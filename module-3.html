Filtering Citra Digital Pemahaman Spasial dan Frekuensi

Filtering Citra Digital: Pemahaman Spasial dan Frekuensi

1. Pendahuluan Filtering Citra Digital

Filtering citra digital adalah teknik dasar dalam image processing yang digunakan untuk memodifikasi kualitas citra. Modifikasi ini dapat berupa:

  • mengurangi noise,
  • menghaluskan citra,
  • menonjolkan detail,
  • atau menyiapkan citra untuk proses lanjutan seperti segmentasi dan pengenalan objek.

Definisi Filtering
Filtering merupakan proses manipulasi nilai intensitas piksel dengan memanfaatkan nilai piksel itu sendiri serta piksel di sekitarnya menggunakan fungsi matematis tertentu. Artinya, setiap piksel baru dihitung berdasarkan konteks lokal atau karakteristik global citra.

Tujuan Filtering

  • Menghilangkan gangguan (noise)
  • Menghaluskan citra (smoothing)
  • Mendeteksi tepi (edge detection)
  • Meningkatkan kontras dan detail
  • Memodifikasi karakteristik visual citra sesuai kebutuhan aplikasi

Domain Filtering

  • Domain spasial: manipulasi dilakukan langsung pada nilai piksel.
  • Domain frekuensi: manipulasi dilakukan setelah citra ditransformasikan ke domain frekuensi menggunakan Transformasi Fourier.

2. Konsep Dasar Filtering Spasial

Filtering spasial bekerja secara lokal, yaitu dengan memproses piksel berdasarkan piksel tetangganya.

Konvolusi
Konvolusi adalah operasi matematika inti dalam filtering spasial. Konsepnya adalah:

  • sebuah kernel digeser ke seluruh citra,
  • nilai piksel di bawah kernel dikalikan dengan bobot kernel,
  • hasilnya dijumlahkan untuk menghasilkan piksel baru.

Kernel / Mask / Filter
Kernel adalah matriks kecil (biasanya 3×3, 5×5, atau 7×7) yang berisi bobot. Bentuk dan nilai kernel menentukan efek filter, misalnya menghaluskan atau menajamkan citra.

Padding
Padding digunakan untuk menangani piksel tepi. Tanpa padding, ukuran citra bisa berkurang. Padding umum:

  • zero padding (nilai 0),
  • replicate (menyalin nilai tepi),
  • reflect (mencerminkan nilai tepi).

3. Konsep Konvolusi Secara Matematis

Pada operasi konvolusi:

  • g(x, y) adalah hasil filtering,
  • f(x, y) adalah citra asli,
  • h(s, t) adalah kernel,
  • ukuran kernel ditentukan oleh parameter tertentu.

Tahapan Konvolusi

  1. Pusat kernel ditempatkan di piksel target
  2. Setiap nilai piksel dikalikan dengan nilai kernel
  3. Hasil perkalian dijumlahkan
  4. Proses diulang ke seluruh piksel citra

Konvolusi memungkinkan citra “diinterpretasi ulang” sesuai tujuan filter.

4. Jenis-Jenis Filter Spasial

a. Low-Pass Filter (Smoothing / Blurring)

Filter ini melemahkan frekuensi tinggi (detail dan noise) dan mempertahankan frekuensi rendah.

Contoh:

  • Average filter: merata-ratakan nilai piksel
  • Gaussian filter: pembobotan mengikuti distribusi Gaussian sehingga hasil lebih alami

Efek utama:

  • citra menjadi lebih halus,
  • detail kecil dan noise berkurang.

b. High-Pass Filter (Sharpening / Edge Detection)

Filter ini menekan frekuensi rendah dan menonjolkan perubahan intensitas.

Contoh:

  • Sobel
  • Prewitt
  • Laplacian

Efek utama:

  • tepi objek lebih jelas,
  • tekstur dan detail meningkat,
  • noise juga bisa ikut diperkuat.

c. Band-Pass Filter

Melewatkan hanya rentang frekuensi tertentu. Digunakan untuk:

  • isolasi fitur khusus,
  • analisis pola tertentu dalam citra.

5. Implementasi Filtering Spasial dengan Python

Implementasi menggunakan OpenCV dan NumPy menunjukkan bahwa:

  • filtering spasial relatif mudah diimplementasikan,
  • cocok untuk real-time processing,
  • parameter kernel sangat memengaruhi hasil.

Perbandingan hasil:

  • Average filter → blur merata
  • Gaussian filter → blur lebih halus dan natural
  • Sobel & Laplacian → menonjolkan tepi

6. Analisis Efek Filtering Spasial

Efek Low-Pass Filter

  • Mengurangi noise acak
  • Menghaluskan citra
  • Mengaburkan detail halus
  • Cocok sebagai tahap pra-pemrosesan

Efek High-Pass Filter

  • Menekankan perubahan intensitas
  • Mendeteksi tepi dan kontur
  • Meningkatkan detail
  • Rentan meningkatkan noise

Pemilihan filter harus disesuaikan dengan tujuan aplikasi.

7. Filtering pada Domain Frekuensi

Filtering frekuensi dilakukan dengan langkah:

  1. Transformasi citra ke domain frekuensi (DFT/FFT)
  2. Penerapan filter pada spektrum frekuensi
  3. Transformasi balik ke domain spasial (IDFT/IFFT)

Keunggulan utama:

  • lebih fleksibel dalam desain filter,
  • efektif untuk filter berukuran besar.

8. Transformasi Fourier

Transformasi Fourier memecah citra menjadi komponen frekuensi.

Pemahaman Domain Frekuensi

  • Frekuensi rendah → area homogen
  • Frekuensi tinggi → tepi, detail, noise
  • Komponen DC → rata-rata intensitas citra

Spektrum frekuensi membantu memahami karakteristik citra sebelum merancang filter.

9. Jenis Filter Frekuensi

  • Ideal Low-Pass: cutoff tajam, bisa menimbulkan artefak (ringing)
  • Butterworth Low-Pass: transisi lebih halus
  • Gaussian Low-Pass: transisi paling halus dan stabil
  • High-Pass Filter: kebalikan low-pass, menonjolkan detail

10. Perbandingan Filtering Spasial dan Frekuensi

  • Filtering spasial lebih intuitif dan sederhana
  • Filtering frekuensi lebih fleksibel dan kuat untuk filter kompleks
  • Spasial efisien untuk kernel kecil
  • Frekuensi lebih efisien untuk kernel besar

Keduanya saling melengkapi dalam pipeline pemrosesan citra.

11. Studi Kasus

a. Peningkatan Kualitas Citra Medis

  • Gaussian filter untuk mereduksi noise
  • Filtering frekuensi untuk menonjolkan struktur anatomi
  • Kombinasi filtering menghasilkan citra yang lebih informatif secara klinis

b. Deteksi Tepi untuk Segmentasi

  • Pra-pemrosesan dengan Gaussian blur
  • Deteksi tepi dengan Sobel atau Canny
  • Morfologi untuk memperbaiki hasil segmentasi

12. Filter Bilateral

Filter bilateral adalah filter non-linear yang:

  • menghaluskan area homogen,
  • tetap mempertahankan tepi tajam.

Parameter:

  • d: diameter tetangga
  • sigmaColor: sensitivitas perbedaan intensitas
  • sigmaSpace: sensitivitas jarak spasial

Filter ini sangat cocok untuk aplikasi visual yang membutuhkan hasil natural.
