Pembelajaran Mesin Lanjutan dalam Computer Vision

Pembelajaran Mesin Lanjutan dalam Computer Vision

1. Pengantar Pembelajaran Mesin

Apa itu Pembelajaran Mesin?

Pembelajaran mesin (machine learning) adalah cabang kecerdasan buatan yang memungkinkan komputer belajar dari data tanpa harus diprogram dengan aturan eksplisit. Alih‑alih menuliskan logika langkah demi langkah, kita menyediakan data dan algoritma akan menemukan pola secara otomatis.

Inti konsepnya:

  • Sistem belajar dari pola data
  • Model semakin baik seiring bertambahnya data
  • Mampu melakukan generalisasi pada data baru

Dalam konteks computer vision, pembelajaran mesin memungkinkan komputer memahami isi gambar seperti manusia: mengenali objek, wajah, tulisan, atau pola visual kompleks.

2. Peran Pembelajaran Mesin dalam Computer Vision

Computer vision bertujuan agar komputer dapat memperoleh pemahaman tingkat tinggi dari citra dan video. Pembelajaran mesin merevolusi bidang ini dengan menggeser pendekatan dari:

  • Rule‑based system (aturan manual)
  • ke data‑driven learning (belajar dari data)

Peran utama ML dalam Computer Vision:

  1. Ekstraksi fitur otomatis
    Model (terutama CNN) belajar langsung fitur penting tanpa rekayasa manual.
  2. Klasifikasi & pengenalan objek
    Misalnya mengenali mobil, manusia, atau hewan dalam gambar.
  3. Segmentasi gambar
    Memisahkan objek dan latar belakang pada tingkat piksel.
  4. Deteksi anomali visual
    Digunakan pada inspeksi kualitas industri atau imaging medis.
  5. Pemahaman kontekstual
    Model memahami hubungan antar objek dalam satu gambar.

3. Jenis‑Jenis Pembelajaran Mesin

3.1 Pembelajaran Terawasi (Supervised Learning)

Definisi:
Algoritma belajar dari data berlabel, yaitu pasangan data input dan target output.

Bentuk data:

(x1,y1),(x2,y2),…,(xn,yn)(x_1, y_1), (x_2, y_2), \dots, (x_n, y_n)(x1​,y1​),(x2​,y2​),…,(xn​,yn​)

Jenis masalah:

  • Klasifikasi → label diskrit (misalnya: kucing/anjing)
  • Regresi → nilai kontinu (misalnya estimasi usia)
  • Peringkat & struktural → urutan atau struktur kompleks

Kelebihan:

  • Evaluasi kinerja jelas
  • Akurasi tinggi jika label berkualitas

Kekurangan:

  • Membutuhkan banyak data berlabel (mahal & lama)

3.2 Algoritma Supervised Learning

  1. K‑Nearest Neighbors (KNN)
    Menentukan kelas berdasarkan mayoritas tetangga terdekat.
    Cocok untuk dataset kecil, namun lambat pada data besar.
  2. Support Vector Machine (SVM)
    Mencari pembatas optimal antar kelas.
    Handal di dimensi tinggi dan data non‑linier.
  3. Decision Tree & Random Forest
    Model berbasis aturan pohon.
    Mudah dipahami, Random Forest lebih stabil dan akurat.
  4. Neural Networks
    Sangat kuat untuk visi komputer.
    Membutuhkan data dan komputasi besar.

3.3 Pembelajaran Tidak Terawasi (Unsupervised Learning)

Definisi:
Algoritma belajar dari data tanpa label, untuk menemukan pola tersembunyi.

Tujuan utama:

  • Memahami struktur data
  • Mengelompokkan atau mereduksi dimensi fitur

Jenis masalah:

  • Clustering
  • Reduksi dimensi
  • Deteksi anomali
  • Pembelajaran representasi

3.4 Algoritma Unsupervised Learning

  1. K‑Means
    Mengelompokkan data berdasarkan jarak. Sederhana dan cepat.
  2. Hierarchical Clustering
    Membentuk hirarki cluster, cocok untuk eksplorasi pola.
  3. PCA (Principal Component Analysis)
    Mereduksi dimensi sambil mempertahankan variansi besar.
  4. Autoencoder
    Neural network untuk representasi fitur non‑linier.

5. Pembelajaran Penguatan (Reinforcement Learning)

Konsep Dasar

Agen belajar melalui interaksi dengan lingkungan untuk memaksimalkan reward kumulatif.

Komponen utama:

  • Agen
  • Lingkungan
  • State
  • Action
  • Reward

Aplikasi dalam Computer Vision:

  • Navigasi visual robot
  • Visual attention mechanism
  • Optimasi proses vision secara adaptif

6. Metrik Evaluasi Pembelajaran Mesin

Metrik Klasifikasi

  • Akurasi
  • Presisi
  • Recall
  • F1‑Score
  • ROC‑AUC

Metrik Regresi

  • MSE, RMSE
  • MAE
  • R²

Metrik Clustering

  • Silhouette Score
  • Davies‑Bouldin Index
  • Calinski‑Harabasz Index

Pemilihan metrik harus sesuai tujuan dan karakteristik masalah.

7. Implementasi dengan Python (Scikit‑learn)

Scikit‑learn menyediakan:

  • Algoritma ML
  • Preprocessing
  • Evaluasi model

Alur umum:

  1. Split data
  2. Normalisasi fitur
  3. Training model
  4. Prediksi
  5. Evaluasi hasil

8. Studi Kasus: Pengenalan Wajah

Pendekatan Supervised

  • Dataset wajah berlabel
  • CNN atau SVM
  • Evaluasi dengan akurasi

Pendekatan Unsupervised

  • Dataset tanpa label
  • Clustering wajah
  • Menemukan pola demografis

9. Dataset Populer Computer Vision

  • ImageNet
  • COCO
  • CIFAR‑10/100
  • Pascal VOC
  • LFW (Faces)

Dataset besar mendorong performa, tetapi menimbulkan tantangan anotasi.

10. Tantangan Pembelajaran Mesin dalam Computer Vision

  • Variasi intra‑kelas
  • Representasi fitur
  • Kebutuhan data besar
  • Generalisasi ke data nyata

11. Preprocessing Data Citra

Preprocessing meningkatkan kualitas pembelajaran melalui:

  • Normalisasi
  • Resize
  • Augmentasi
  • Filtering
  • Histogram equalization
