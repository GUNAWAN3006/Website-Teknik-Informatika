Convolutional Neural Network

Convolutional Neural Network (CNN)

1. Pengantar Convolutional Neural Network (CNN)

Convolutional Neural Network (CNN) adalah jenis Deep Neural Network yang dirancang khusus untuk memproses data berbentuk grid, terutama citra digital. CNN terinspirasi dari cara kerja visual cortex pada manusia, di mana neuron merespons area lokal tertentu pada medan visual.

Mengapa CNN penting dalam Computer Vision?

  • Citra memiliki struktur spasial (hubungan antar piksel).
  • CNN mampu mempelajari pola lokal → global secara bertahap.
  • Jauh lebih efisien dibanding neural network biasa untuk citra.

Keunggulan CNN:

  • Ekstraksi fitur otomatis (tidak perlu hand-crafted feature).
  • Weight sharing → jumlah parameter lebih sedikit.
  • Relatif tahan terhadap pergeseran objek (translation invariance).

Aplikasi utama CNN:

  • Klasifikasi citra
  • Deteksi objek
  • Segmentasi citra
  • Pengenalan wajah

2. Arsitektur Dasar CNN

CNN bekerja dengan menumpuk beberapa jenis lapisan, dari input hingga output. Alurnya bersifat hierarkis.

a. Input Layer

  • Menerima citra sebagai tensor 3D:
    tinggi × lebar × channel
  • Contoh: citra RGB CIFAR‑10 → 32 × 32 × 3

b. Convolutional Layer

  • Melakukan operasi konvolusi dengan filter/kernel
  • Menghasilkan feature map
  • Lapisan awal → fitur sederhana (tepi, sudut)
  • Lapisan dalam → fitur kompleks (tekstur, objek)

c. Pooling Layer

  • Mereduksi dimensi spasial
  • Mengurangi komputasi
  • Membuat model lebih stabil terhadap translasi

d. Fully Connected Layer

  • Menggabungkan seluruh fitur
  • Digunakan untuk proses klasifikasi akhir

3. Convolution Layer: Konsep Inti

Convolution layer menggeser kernel di atas citra input dan menghitung operasi perkalian + penjumlahan.

Inti proses:

  1. Kernel mengambil area kecil citra
  2. Nilai dikalikan bobot
  3. Dijumlahkan → satu nilai feature map
  4. Kernel digeser (stride)

Makna Feature Map

  • Setiap feature map merepresentasikan satu jenis pola
  • Banyak filter → banyak sudut pandang fitur

4. Parameter Penting pada Convolution Layer

a. Filter Size (Kernel)

  • Umum: 3×3, 5×5
  • Kernel kecil berlapis lebih efektif daripada kernel besar

b. Stride

  • Jumlah pergeseran kernel
  • Stride besar → output lebih kecil → informasi bisa hilang

c. Padding

  • Valid: tanpa padding → ukuran mengecil
  • Same: padding → ukuran dipertahankan

d. Jumlah Filter

  • Menentukan banyaknya feature map
  • Lebih banyak filter → lebih kaya fitur, tapi lebih berat komputasi

5. Fungsi Aktivasi dalam CNN

Fungsi aktivasi menambahkan non-linearitas, memungkinkan CNN mempelajari pola kompleks.

ReLU (Rectified Linear Unit)

f(x) = max(0, x)

✅ Cepat, sederhana
⚠️ Risiko dying ReLU

Leaky ReLU

f(x) = max(αx, x)

✅ Mengatasi dying ReLU

Sigmoid & Tanh

  • Jarang digunakan di hidden layer
  • Masih dipakai pada output klasifikasi biner

6. Pooling Layer: Pengurangan Dimensi

a. Max Pooling

  • Mengambil nilai maksimum
  • Menjaga fitur dominan
  • Paling sering digunakan

b. Average Pooling

  • Mengambil nilai rata-rata
  • Menjaga konteks global

Dampak Pooling: ✅ Mengurangi overfitting
✅ Mempercepat pelatihan
❌ Kehilangan detail spasial

7. Fully Connected Layer & Output

Flattening

  • Feature map 3D → vektor 1D
    Contoh: 7×7×64 = 3136 elemen

Fully Connected

  • Menggabungkan seluruh fitur
  • Biasanya menggunakan ReLU

Output Layer

  • Jumlah neuron = jumlah kelas
  • Softmax → probabilitas tiap kelas

8. Teknik Regularisasi CNN

Dropout

  • Mematikan neuron secara acak
  • Mencegah ketergantungan berlebih

Batch Normalization

  • Menstabilkan distribusi aktivasi
  • Mempercepat konvergensi

Data Augmentation

  • Rotasi, flipping, scaling
  • Menambah variasi data tanpa data baru

9. Arsitektur CNN Populer

  • LeNet → cikal bakal CNN
  • AlexNet → revolusi deep learning (ILSVRC 2012)
  • VGG → arsitektur dalam dan seragam
  • ResNet → skip connection, sangat dalam

10. Transfer Learning

Keuntungan:

  • Hemat data
  • Cepat konvergen
  • Performa tinggi

Pendekatan:

  1. Feature extraction
  2. Fine-tuning
  3. Full training

11. Studi Kasus: CIFAR‑10

  • 60.000 citra (32×32, RGB)
  • 10 kelas objek

Hasil utama:

  • CNN sederhana → akurasi 75–80%
  • Data augmentation → +3–5%
  • Layer konvolusi tambahan → signifikan

12. Tantangan CNN

  • Membutuhkan data besar
  • Komputasi tinggi (GPU)
  • Sensitif hyperparameter
  • Interpretabilitas rendah (black box)

13. Tren Terkini

  • Capsule Networks
  • EfficientNet
  • Vision Transformer (ViT)
  • CNN + Attention

14. Ringkasan Pembelajaran

✅ CNN adalah fondasi utama computer vision modern
✅ Lapisan konvolusi = inti kekuatan CNN
✅ Transfer learning penting untuk aplikasi nyata
✅ Pemahaman arsitektur lebih penting daripada sekadar menjalankan kode
